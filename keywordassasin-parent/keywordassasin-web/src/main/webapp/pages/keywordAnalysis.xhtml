1. Go to google keyword table, make a search on desired phrase and export the result in CSV format. 2. Press the choose button and locate your CSV file. Use the upload button to upload it to server. 3. Press the start button to start analyzing. You can pause and resume the analysis. The analysis dynamically updates a bar chart, showing the distribution of the keywords accross different rating categories. 4. Use the download button at any time during analysis to download the result. Few Tips: 1. Use maximum 100 keywords in one anaysis cycle. 2. Make sure the downloaded file from google is comma seperated. 3. Make sure you have these 3 columns in CSV file; keyword, CPC, global monthly searches
